# راهنمای اجرای بات سنایی روی کلودفلر ورکر (بدون VPS)

## پیش‌نیازها
۱. حساب کاربری در کلودفلر (Cloudflare).
۲. نصب بودن Node.js و npm روی سیستم شما (برای دیپلوی).
۳. ایجاد یک فضای ذخیره‌سازی KV در کلودفلر (برای دیتابیس بدون سرور).

## مراحل دیپلوی بک‌اند روی Cloudflare Workers

1. یک KV Namespace در پنل کلودفلر خود (از بخش **Workers & Pages** > **KV**) با نام دلخواه (مثلاً `sanaei_db`) بسازید و ID آن را کپی کنید.

2. فایل `wrangler.toml` را در پوشه اصلی پروژه باز کرده و `id = "..."` را با شناسه (ID) ذخیره‌سازی KV خود جایگزین کنید.

3. دستور زیر را برای ورود به حساب کلودفلر خود و تایید دسترسی اجرا کنید:
\`\`\`bash
npx wrangler login
\`\`\`

4. حال برای انتشار (Deploy) ربات دستور زیر را بزنید:
\`\`\`bash
npx wrangler deploy
\`\`\`

5. پس از دیپلوی، یک لینک ورکر شبیه `https://sanaei-bot-worker.<your-subdomain>.workers.dev` دریافت می‌کنید. 

6. حال برای اینکه تلگرام آپدیت‌ها را به ورکر شما بفرستد (وب‌هوک سرورلس)، باید آدرس زیر را در مرورگر وارد کنید تا وب‌هوک تنظیم شود:
\`https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://sanaei-bot-worker.<your-subdomain>.workers.dev/api/webhook/telegram\`
(به جای `<TOKEN>`، توکن ربات تلگرامی خود را بگذارید).

## دیپلوی پنل ادمین (فرانت‌اند) روی Cloudflare Pages
بات شما اکنون به طور کامل روی Worker و بدون نیاز به VPS کار می‌کند. 
برای مشاهده پنل ادمین وب (بخش فرانت‌اند)، کافیست با دستور زیر آن را روی Cloudflare Pages دیپلوی کنید:

\`\`\`bash
npm run build
npx wrangler pages deploy dist --project-name sanaei-panel
\`\`\`
پس از آن، آدرس پنل به شما داده می‌شود که می‌توانید وارد آن شوید. حتما دقت کنید که در تنظیمات پنل ادمین، آدرس API را همان آدرس Worker خود قرار دهید.

**تبریک! ربات سنایی و پنل آن حالا به صورت ۱۰۰٪ بدون VPS و روی زیرساخت Cloudflare در حال اجرا است.**
